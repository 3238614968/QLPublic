# QLPublic

个人存档备份。当前仓库主要包含一个用于品赞 HTTP 代理每日签到的 Python 脚本。

## 目录结构

- `品赞签到.py`：主脚本，支持多账号登录并执行签到。

## 功能说明

脚本会按以下流程执行：

1. 从环境变量 `pzhttp` 读取账号配置。
2. 对每个账号执行登录，获取 token。
3. 使用 token 调用签到接口。
4. 输出每个账号的执行结果与汇总信息。

## 环境变量配置

变量名：`pzhttp`

格式：`账号#密码`，多账号用换行分隔。

示例：

```text
13800138000#MyPass123
13900139000#AnotherPass456
```

## 运行方式

### 本地运行

1. 安装 Python 3.8+。
2. 安装依赖：

```bash
pip install requests
```

3. 配置环境变量并运行：

```bash
export pzhttp=$'13800138000#MyPass123\n13900139000#AnotherPass456'
python 品赞签到.py
```

### 青龙面板

1. 新建环境变量 `pzhttp`，按上面的格式填写。
2. 新建定时任务执行：

```bash
python 品赞签到.py
```

## 输出说明

- `登录失败`：账号密码错误、风控、或接口返回失败信息。
- `签到失败`：登录成功但签到接口返回失败。
- `登录请求异常 / 签到请求异常`：网络问题、超时、依赖缺失等。

## 注意事项

- 请勿在公开场景泄露账号密码。
- 接口属于第三方服务，若页面或接口策略调整，脚本可能需要同步更新。
- 如环境缺少 `requests`，脚本将无法运行。
